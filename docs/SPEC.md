# Sunday Play — 기획서

## 프로젝트 개요

- **프로젝트명:** Sunday Play
- **한 줄 설명:** 교회 레크레이션 큐레이션 플랫폼
- **목표:** 구독제를 통한 수익화
- **타겟 사용자:** 주일학교 교사, 청년부 리더, 교역자 등 주일을 위해 게임을 준비해야 하는 크리스천들

---

## 핵심 기능

### 1. 게임/활동 목록 (데이터 모델 및 뷰)

사용자가 리스트에서 '직관적'으로 판단하고, 상세 페이지에서 '사역'에 바로 적용하게 만드는 구조.

- **카드 뷰 (Summary Card):** 한눈에 보는 정보
  - **콘텐츠:** 게임명, 썸네일, 핵심 태그(대상, 공간 등)
  - **메타데이터:** 소요 시간, 권장 인원, 준비물 유무(아이콘) 등
- **상세 정보 (Detailed Modal):** 클릭 시 나타나는 상세 가이드
  - **기본 정보:** 준비물 리스트, 난이도(1~5단계), 상세 진행 스텝
  - **Sunday Play 특화:**
    - **말씀 연결:** 연관 성경 구절 + 아이들에게 전할 1분 메시지 요약
    - **진행 스크립트:** 한/영 대사 예시
    - **변형 팁:** 인원이 적거나 장소가 좁을 때의 대안

### 2. 필터 및 검색 (Search & Taxonomy)

교사가 처한 '상황'에 따라 가장 빠르게 검색 결과에 도달하게 설계.

- **대상(Age Group):** 영아부 / 유치부 / 유년부 / 초등부 / 중고등부 / 청년부 / 장년부
- **활동성(Energy Level):** 1단계(정적/앉아서) ~ 5단계(격동적/뛰면서)
- **공간(Environment):** 실내(교실) / 실내(강당) / 야외 / 온라인
- **준비 시간(Prep Time):** 준비물 없음 / 5분 내 준비 / 사전 준비 필요
- **성경 주제(Biblical Themes):** 사랑, 순종, 믿음, 감사, 절제 등 공과 주제별 필터
- **인원수 범위:** 5명 미만 / 5~10명 / 10~30명 / 30명 이상 대그룹

### 3. 사용자 및 커뮤니티 기능 (Engagement)

- **보관함:** 이번 주 주일학교에서 할 게임들을 '내 보관함'에 저장
- **리뷰:** 현장 팁 공유 (예: "7세 아이들에게는 이 규칙이 어려웠어요")
- **간편 공유:** 카카오톡/링크 공유를 통해 보조 교사에게 바로 전달
- **평점 시스템:** 실제 현장에서의 반응도를 기반으로 한 정렬 기능

### 4. 프리미엄 기능 (Subscription) — 수익화 포인트

$5/월 구독 모델.

- **PPT/자료 다운로드:** 퀴즈 게임용 PPT 템플릿, 인쇄용 도안(PDF) 제공
- **위클리 큐레이션:** 이번 주 공과 주제에 맞춘 3종 게임 세트 자동 추천
- **진행 툴킷:** 상세 페이지 내에 '타이머', '랜덤 팀 나누기' 웹 도구 내장

### 5. 현장 모드 (Presenter Mode)

상세 페이지 상단의 버튼을 누르면 폰트 크기가 200% 커지고, 복잡한 UI가 사라지며 다음 단계로 넘기는 큰 버튼과 스톱워치 등만 노출되는 모드.

## 콘텐츠 전략

- **초기 콘텐츠 수:** 20~30개 내외로 시작해 점차 확충
- **콘텐츠 소스:** 직접 작성 + AI 어시스트 (LLM을 이용해 톤앤매너 통일, 성경 구절 매칭 자동화)

---

## 수익화

- **무료 범위:**
  - 전체 게임 목록 브라우징 및 검색
  - 게임의 제목, 썸네일, 핵심 태그, 짧은 요약 설명, 말씀 연결 브릿지
  - 상세 가이드는 월 3개 제한
- **유료 범위 ($5/월):**
  - 모든 상세 가이드 무제한 열람
  - 진행 스크립트 (한/영)
  - 다운로드: PPT 템플릿, 활동지 PDF, 고화질 이미지 소스
  - 진행 도구: 인앱 타이머, 팀 나누기 툴 사용 권한
- **결제:** Stripe
- **주 타겟 시장:** 한국 시장 우선 런칭

## 기술 요구사항

- **기술 스택:** Next.js + Supabase (Free Tier)
- **배포:** Vercel (Free)
- **반응형:** 모바일 대응 필수 (교사들이 현장에서 폰으로 가이드 확인)
- **인증:** 소셜 로그인 (구글) — Supabase Auth 사용

## 우선순위

### MVP (1차)

- 회원가입 및 소셜 로그인
- 게임 리스트 필터/검색 기능
- 기본 상세 페이지 (말씀 연결 포함)
- 모바일 반응형 레이아웃

### 2차 (수익화)

- 정기 구독 결제 시스템 연동
- 자료(PDF/PPT) 다운로드 기능
- 영어 스크립트 데이터 확충
- 다국어 지원

### 3차 (고도화)

- 진행 보조 툴킷 (타이머 등) 구현
- AI 기반 이번 주 공과 맞춤 게임 추천 엔진
- 사용자 참여형 커뮤니티 및 리뷰 시스템
